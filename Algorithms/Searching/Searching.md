# &#128161; What is the Searching
**Searching** is a widely used operation in computer science to retrieve information. It helps to locate specific items or data within datasets in various algorithms and for all manner of applications, therefore it is important to truly understand the concept.
<p align="center"><img src="./img/searching.png"/></p>

---
**Essential Terminology:**
- **Searching Algorithm** â€” is a set of steps used to retrieve a particular element within a collection of data.
  - **Successful Search** â€” is an outcome in which algorithm found the element(-s) containing argument. 
  - **Unsuccessful Search** â€” is an outcome in which algorithm didn't the element(-s) containing argument.
- **Collection** â€” is a data structure, which is used to group multiple elements together.
  - **Table** â€” is a small collection of all elements.
  - **File** â€” is a big collection of all elements.
  - **Database** â€” is a large collection of all elements.
- **Element/Record/Item** â€” is an individual entity within a collection.
  - **Index/Key of Element** â€” is a position of an element within collection.
  - **Value/Data of Element** â€” is an actual information stored within the element.
- **Argument** â€” is a parameter passed to an algorithm, which provides the criteria, which will be used to find the desired element.
- **Criterion** â€” is a rule used to determine whether an element meets the requirements of the search.
